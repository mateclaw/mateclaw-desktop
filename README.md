  # MateClaw

  **OpenClaw 企业版 · 超级数字员工系统**

  MateClaw 是一款面向企业的 AI 数字员工平台，支持多渠道接入（飞书、钉钉、企业微信、Telegram 等），让 AI 员工真正融入企业协作流程。

  ## 功能特性

  - 多平台渠道接入：飞书、钉钉、企业微信、Telegram、WhatsApp
  - 数字员工管理：自定义员工人格、记忆、技能
  - 定时任务：支持 Cron 调度，自动执行任务
  - MCP 工具扩展：灵活接入各类外部工具
  - 本地运行：数据私有，安全可控

  - macOS 11.0+（Apple Silicon / Intel）
  - Windows 10+
