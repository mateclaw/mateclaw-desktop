## 产品定位

MateClaw 是一套面向企业的 **AI 数字员工平台**。不同于普通 AI 助手，MateClaw 中的每个数字员工都拥有独立的角色身份、记忆系统、技能库和企业工具接入能力，能够真正替代人工完成钉钉审批跟进、飞书文档整理、邮件归类回复、数据库查询分析、浏览网页、定时自动化任务等重复性工作。

**核心理念：AI 不只是聊天框，而是一位真正上岗的数字同事。**

员工技能热加载，安装即用，支持自定义开发。内置技能涵盖：

- 浏览器自动化（支持 Shadow DOM、复杂 SPA）

- 网页内容提取与搜索

### 员工权限管控：AI 能干什么，你说了算

这是 MateClaw 与市面上其他 AI 工具最核心的差异之一。每个数字员工都有一套独立的权限策略，明确界定它被允许做什么、什么情况下需要人工审批、什么是绝对禁止的。

**三态权限模型（auto / human / deny）：**

| 能力                | 说明                                   |
| ------------------- | -------------------------------------- |
| 命令执行      | 默认需人工审批，可配置为自动或禁止     |
| 工作目录内文件写入  | 默认自动，员工可在指定范围内自主操作   |
| 工作目录外写入      | 默认需人工审批，敏感路径可设为永久保护 |
| 工作目录外读取      | 默认自动，可限制为仅允许特定路径       |
| 浏览器访问          | 三级：禁用 / 只读观察 / 可交互操作     |
| 单工具 / MCP 工具 / 插件 工具 | 可为每个工具单独设置审批策略           | 

**运行时实时授权：** 当员工尝试执行需要审批的操作时，界面弹出授权请求，显示完整的工具调用内容，人类可选择「允许一次」「本会话允许」或「拒绝」。整个执行过程在人类的可见、可控范围内进行。

**受保护路径：** 可设定关键目录为保护区，命中后必须人工审批，且不允许被会话规则放宽——即使员工在同一会话内已被放行过其他操作。

这套机制让企业可以在享受 AI 自动化效率的同时，对敏感操作保持完整的掌控权，不必担心 AI"越权"。

## 云端平台：企业级统一管控中心

桌面客户端让单个员工"能用起来"，而云端平台解决的是企业级问题：**谁在用、用了什么、怎么管、怎么控**。两者配合构成完整的企业 AI 数字员工体系。

### 人类员工 × 数字员工：统一管理

企业的真实工作由人与 AI 共同完成。云端平台将两类员工放在同一视图下统一管理：

- **人类员工**：组织架构、岗位、权限角色绑定，是数字员工服务的对象
- **数字员工**：角色定义、技能配置、渠道绑定、任务记录全面可见
- 每个数字员工归属于具体的人类员工或团队，职责清晰，不存在"没人负责"的 AI

### 企业资产中台：六类核心资产统一沉淀

企业积累的 AI 能力不再散落在每个人的电脑里，而是沉淀为可共享、可复用的企业资产：

| 资产类型     | 说明                                             |
| ------------ | ------------------------------------------------ |
| **知识库**   | 企业文档、规章制度、产品手册，数字员工的知识来源 |
| **技能库**   | 企业自研或从市场安装的技能包，员工按需取用       |
| **数据库**   | 接入企业内部数据库，员工可查询、理解数据         |
| **知识图谱** | 实体关系图谱，帮助 AI 理解复杂的业务关联         |
| **插件库**   | 扩展能力的功能插件，统一管理与分发               |
| **人才库**   | 从市场引入或自主创建的数字员工模板，随时部署     |

### 人才市场：买即用，用即上岗

云端维护一个开放的数字员工市场。企业无需从零构建，直接选购适合岗位的数字员工：

- 浏览、试用、订阅（月付 / 年付 / 买断）
- 下载安装到本地客户端，版本与配置自动同步
- 员工有最低客户端版本要求，兼容性问题提前拦截
- 发布者可提交审核，审核通过后上架，全程可追溯

### 企业权限与安全

权限不只在员工侧，云端同样建立了一套完整的企业级权限体系：

- **RBAC 角色权限**：按角色分配功能访问权，不同运营人员只看到各自负责的模块
- **证书颁发与管控**：精确控制哪个组织可以使用哪些功能、使用到何时
- **订阅状态同步**：员工订阅关系以云端为准，卸载重装不丢失，换设备无缝继续
- **审核流程**：数字员工上架需经过人工审核，内容与行为边界有明确把控
- **操作日志**：执行记录、任务中心全面留存，企业合规有据可查
